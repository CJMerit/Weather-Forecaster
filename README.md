## Description
Want to take a trip? You need to plan for the weather, wherever you're going! 
Just put in the city and zip code you are visiting, and you'll get the forecast for the next five days!

## Built With
- HTML
- CSS
- Javascript
- JQuery
- Bootstrap
- Day.js
- OpenWeather API
